## 1과목 텔러기본지식(27문항)
 #### 1장 금융경제일반
   * 통화-금리-금융시장
    - 통화의 기능은 교환의 매개/가치저장/가치척도/이연지급이다.
      이연지급(장래에 지급해야 할 채무의 표시수단)
    - 통화는 금본위제 -> 관리통화체제로 전환 됐다.
      금본위제 -> 금화를 돈으로  
      관리통화체제 -> 당국이 통화 공급량 통제하는 제도
     - 통화 및 유동성 지표
       1. 통화지표  
         협의통화(M1) : 단기금융시장의 유동성 수준을 파악하는 지표  
                        ex) 현금 + 요구불예금
